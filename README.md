# Teleprompter Web

Herramienta de teleprompter basada en web, diseñada para facilitar la grabación de videos y presentaciones. Funciona completamente en el navegador sin necesidad de instalación.

## Características Principales

-   **Control de Reproducción**: Inicio/Pausa con barra espaciadora o botón en pantalla.
-   **Velocidad Ajustable**: Control preciso de la velocidad de desplazamiento (1x - 10x).
-   **Tamaño de Fuente**: Ajuste dinámico del tamaño del texto.
-   **Modo Espejo**: Invierte el texto horizontalmente para su uso con hardware de teleprompter profesional.
-   **Edición en Tiempo Real**: Editor integrado con soporte para sintaxis simplificada y HTML.

## Guía de Uso

### Atajos de Teclado
-   **Espacio**: Iniciar / Pausar.
-   **Flecha Arriba**: Aumentar velocidad.
-   **Flecha Abajo**: Disminuir velocidad.

### Edición del Guion
El editor incluye un menú interactivo "Insertar..." para facilitar el formato:

1.  **Bloques de Sección**:
    -   Usa el botón "Nuevo Bloque" o escribe `[bloque] Título`.
    -   Crea encabezados rojos para separar secciones del guion.
2.  **Variables**:
    -   Usa el botón "Marcador" o escribe `[Nombre]`.
    -   Resalta texto en amarillo para indicar datos que deben personalizarse (ej: `[Tu Nombre]`).

## Tecnologías

-   HTML5
-   Tailwind CSS (vía CDN)
-   React & ReactDOM (vía CDN)
-   Lucide Icons
